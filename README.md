# ✈️ Agência de Viagens - Domínio de Bootstrap

![Status](https://img.shields.io/badge/Status-Finalizado-green)

Este repositório registra meu primeiro contato prático com o **Bootstrap**, sendo um passo fundamental na minha jornada como **Desenvolvedor Full-Stack**. O desafio principal foi construir um site completo de agência de viagens utilizando exclusivamente as classes nativas do framework, sem escrever uma única linha de CSS próprio.

## 🚀 Principais Aprendizados

O desenvolvimento deste projeto permitiu entender o poder da padronização e da agilidade no front-end:

- 🎨 **Desenvolvimento "Zero CSS"**: Aprendi a utilizar as classes de utilitários e componentes do Bootstrap para criar uma interface rica e profissional sem depender de estilos customizados.
- 📱 **Responsividade Nativa**: Explorei o sistema de grid do Bootstrap, o que tornou a tarefa de adaptar o site para diferentes tamanhos de tela (desktop e mobile) muito mais simples e eficiente.
- 🧩 **Uso de Componentes**: Implementação e customização de componentes essenciais para a experiência do usuário (UX), como carrosséis, abas (tabs), cartões (cards) e formulários.
- 📐 **Padronização Visual**: Aplicação consistente de espaçamentos, tipografia e cores seguindo o guia de design do framework.

## 🛠️ Tecnologias Utilizadas

- **Estruturação**: HTML5.
- **Design/UI**: Bootstrap 5 (Framework principal de estilo).
- **Imagens**: Assets otimizados para exibição em carrosséis e cards.
